\doxysubsection{Detailed Description}
Represents the multipart/form-\/data of an HTTP request. 

This class encapsulates the multipart/form-\/data of an HTTP request, including any files uploaded as part of that request. It provides functionality to parse the multipart content and retrieve the files.
