\doxysubsection{Detailed Description}
Class responsible for parsing configuration files to construct \mbox{\hyperlink{classServerConfig}{Server\+Config}} objects. 

The \mbox{\hyperlink{classParser}{Parser}} opens and reads a configuration file, interpreting its content to construct and return instances of \mbox{\hyperlink{classServerConfig}{Server\+Config}} that represent the server configurations found within the file.
