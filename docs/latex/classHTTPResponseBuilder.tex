\doxysubsection{Detailed Description}
A builder class that constructs a raw HTTP response string from an \mbox{\hyperlink{classHTTPResponse}{HTTPResponse}} object. 

This class provides static methods to convert an \mbox{\hyperlink{classHTTPResponse}{HTTPResponse}} object into a raw HTTP response string. It handles the creation of the status line, headers, and assembles them with the response body based on the server configuration and the specific HTTP request received.
