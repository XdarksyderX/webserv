\doxysubsection{Detailed Description}
A parser class that interprets raw HTTP requests. 

This class provides static methods to parse raw HTTP request strings into an \mbox{\hyperlink{classHTTPRequest}{HTTPRequest}} object. It processes the request line and headers and populates the \mbox{\hyperlink{classHTTPRequest}{HTTPRequest}} object with parsed data.
