\doxysubsection{Detailed Description}
\mbox{\hyperlink{classLocationConfig}{Location\+Config}} class holds the configuration specific to a location (URI path) within a server. 

This class encapsulates details such as the root directory for the location, allowed methods, default index file, autoindex setting, and CGI configuration.
