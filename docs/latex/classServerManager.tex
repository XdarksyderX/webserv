\doxysubsection{Detailed Description}
Class that manages multiple HTTP servers. 

This class is designed to manage multiple HTTP servers, allowing to start and stop them all at once. It also provides functionalities to add and remove client socket file descriptors to the servers. The class uses the poll system call to handle multiple servers and clients I/O operations simultaneously.
